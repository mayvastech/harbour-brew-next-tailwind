# Harbour Brew Coffee — Portfolio Case Study

## Project Type

Fictional coffee shop website built as a portfolio demonstration project.

## Goal

The goal was to design and build a modern, mobile-friendly website for an independent coffee shop. The site needed to present the brand clearly, show the menu, communicate opening hours and make it easy for visitors to find or contact the business.

## Target User

The target user is someone looking for a local coffee shop where they can quickly check the menu, opening times, location and overall atmosphere before visiting.

## Problem

Many small local businesses have websites that feel outdated, difficult to use on mobile, or unclear about basic information such as opening times, menu items and location.

## Solution

I built a clean one-page website with clear navigation, strong visual sections, responsive layout, menu cards, business information and a simple contact area.

## Key Features

- Sticky navigation
- Responsive mobile-first layout
- Hero section with clear call-to-action
- Menu section
- Brand story section
- Visit/contact information
- Modern visual design
- Portfolio demo disclaimer

## Tech Stack

- Next.js
- Tailwind CSS
- React components
- Vercel-ready deployment

## What I Learned

This project helped me practise building a full landing page using reusable components, responsive layouts, structured content and modern frontend styling.
